**Concept:**
You can update all elements in a list using a loop. You can also use the `*` operator to multiply each element. Example:
```python
arr = [2, 4, 6]
for i in range(len(arr)):
    # update each element by multiplying by 2 
    arr[i] = arr[i] * 2 
print(arr)  # prints [4, 8, 12]
```
