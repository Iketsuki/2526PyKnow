**Concept:**
You can print all elements in a list using a `for` loop or a `while` loop.
- For loop:
```python
arr = [1, 2, 3]
for x in arr:
    print(x)
```
- While loop:
```python
arr = [1, 2, 3]
i = 0
while i < len(arr):
    print(arr[i])
    i += 1
```
