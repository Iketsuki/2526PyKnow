# Python - Strings - String Formatting

## Concept
Use f-strings for clean formatting: `f'text {variable}'` or older `'text {}'.format(var)`.

## Example Code
```python
name = 'Alice'
age = 15
print(f'{name} is {age} years old.')
print('{} is {} years old.'.format(name, age))
```
