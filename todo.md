Brainstorm: Atomic List-Focused Task Ideas

Input + Access

Task: Read several items into a list, then print the n-th item.
Skill: Storing and accessing elements by index.
Input + Update

Task: Read a list of numbers, then update the value at a given position and print the updated list.
Skill: Modifying list elements.
Input + Sum

Task: Read a list of numbers, then print the sum of all elements.
Skill: Iterating over a list and aggregating values.
Input + Search

Task: Read a list of words, then check if a given word is in the list and print “Yes” or “No”.
Skill: Membership testing in a list.
Input + Count

Task: Read a list of numbers, then count how many times a given number appears.
Skill: Counting occurrences in a list.
Input + Reverse

Task: Read a list of items, then print them in reverse order.
Skill: Reversing and iterating over a list.
